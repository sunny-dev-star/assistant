# 快递公司识别规则

## 单号前缀规则

| 前缀 | 快递公司 | 单号格式 | 示例 |
|------|----------|----------|------|
| SF | 顺丰速运 | SF + 10-12位数字 | SF1234567890 |
| JD | 京东物流 | JD + 10-13位数字 | JD0012345678 |
| YT | 圆通速递 | YT + 10-13位数字 | YT98765432100 |
| ZTO | 中通快递 | ZTO + 10-13位数字 | ZTO8888888888 |
| STO | 申通快递 | STO + 10-13位数字 | STO7777777777 |
| YD | 韵达快递 | YD + 10-13位数字 | YD6666666666 |
| EMS | EMS | EMS + 9-13位字符 | EMS123456789 |

## 物流状态流转

```
已揽收 → 运输中 → 到达目的城市 → 派送中 → 已签收
```

## 状态说明

| 状态 | 说明 |
|------|------|
| 已揽收 | 快递员已取件 |
| 运输中 | 在转运中心之间运输 |
| 到达目的城市 | 到达目的地城市的转运/配送站 |
| 派送中 | 快递员正在配送 |
| 已签收 | 收件人已签收 |
