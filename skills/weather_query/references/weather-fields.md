# 天气数据字段说明

## get_weather 返回字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| 城市 | string | 查询的城市名 | "北京" |
| 天气 | string | 天气状况描述 | "小雨" |
| 温度 | string | 当前气温 | "18°C" |
| 体感温度 | string | 人体实际感受温度 | "16°C" |
| 湿度 | string | 空气湿度百分比 | "94%" |
| 风速 | string | 风速 | "5 km/h" |
| 风向 | string | 风向（16方位） | "ENE" |
| 能见度 | string | 能见度距离 | "4 km" |
| 气压 | string | 大气压强 | "1014 hPa" |
| 紫外线指数 | string | UV指数（1-11+） | "1" |

## 紫外线指数等级

| 指数 | 等级 | 建议 |
|------|------|------|
| 1-2 | 低 | 无需防晒 |
| 3-5 | 中等 | 建议戴帽涂防晒 |
| 6-7 | 高 | 必须防晒，避免正午出门 |
| 8-10 | 很高 | 尽量避免户外活动 |
| 11+ | 极端 | 不要外出 |

## get_forecast 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 日期 | string | 预报日期 YYYY-MM-DD |
| 天气 | string | 天气状况 |
| 最高温 | string | 当日最高温度 |
| 最低温 | string | 当日最低温度 |
| 平均温度 | string | 当日平均温度 |
