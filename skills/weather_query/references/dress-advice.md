# 穿衣建议规则

根据温度和天气状况给出穿衣建议。

## 按温度区间

| 温度 | 穿衣建议 |
|------|----------|
| ≥30°C | 短袖短裤，轻薄透气面料 |
| 25-29°C | 短袖/薄长袖 |
| 20-24°C | 长袖衬衫、薄外套 |
| 15-19°C | 卫衣/薄毛衣+外套 |
| 10-14°C | 毛衣+厚外套/风衣 |
| 5-9°C | 羽绒服/棉服 |
| <5°C | 厚羽绒服+保暖内衣+帽子手套 |

## 按天气状况

| 天气 | 建议 |
|------|------|
| 雨 | 带伞、防水鞋 |
| 雪 | 保暖+防滑鞋 |
| 雾 | 能见度低，开车慢行 |
| 大风 | 防风外套 |
| 紫外线高 | 防晒霜+墨镜 |

## 综合建议示例

- 18°C + 小雨 → 薄外套 + 带伞 + 防水鞋
- 26°C + 晴 → 短袖 + 防晒
- 8°C + 大风 → 厚外套 + 帽子
